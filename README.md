# MedLink

MedLink é uma plataforma web desenvolvida para conectar pacientes e médicos por meio de um sistema de agendamento de consultas online.

O objetivo do sistema é facilitar o processo de encontrar profissionais de saúde, visualizar especialidades médicas disponíveis e realizar agendamentos de forma simples e organizada.

Este projeto está sendo desenvolvido como parte da disciplina de **Programação Web**.

---

## Funcionalidades Principais

- Cadastro e autenticação de usuários
- Cadastro de médicos e suas especialidades
- Busca de médicos por especialidade
- Agendamento de consultas entre pacientes e médicos
- Gerenciamento de consultas agendadas
